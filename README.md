# GenAI Course Assignment 1

This repository documents my workflow using a generative AI assistant to create and refine professional content. It focuses on the application of AI in the healthcare industry from my perspective as a JHU student.

## Git Steps Used
To manage this project, I used the following Git workflow:
* `git init` to set up the local repository.
* `git add` to stage my changes for both AI-generated and manually edited files.
* `git commit -m` to document each specific step of the assignment.
* `git push` to synchronize my local progress with the GitHub repository.
